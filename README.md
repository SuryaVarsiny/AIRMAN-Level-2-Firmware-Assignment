# AIRMAN-Level-2-Firmware-Assignment
✅ Performance

Update rate: ~20 Hz telemetry, ~10 Hz visualization.
Runs stably on Windows using MSYS2 + PowerShell.

✅ Limitations

Complementary filter is sensitive to gyro drift.
Magnetometer is simulated, so heading is not absolute.
No sensor calibration implemented.

✅ AI Usage

AI tools (ChatGPT) were used to assist with AHRS math validation, telemetry formatting, and debugging visualization issues.
